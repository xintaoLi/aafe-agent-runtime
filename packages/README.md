# AAFE Publishable Package Plan

This directory contains publishable package placeholders for splitting `@aafe/agent-runtime` into independent packages.

## Core

- `@aafe/core`: runtime, pipelines, gates, memory, config loader

## Framework Packs

- `@aafe/react`
- `@aafe/next`
- `@aafe/vue`

## Scenario Packs

- `@aafe/ddd`
- `@aafe/patterns`
- `@aafe/graph`
- `@aafe/admin`
- `@aafe/dashboard`
- `@aafe/workflow`

## Editor Adapters

- `@aafe/adapter-cursor`
- `@aafe/adapter-claude`

Future adapters can follow the same package shape, e.g. `@aafe/adapter-windsurf`, `@aafe/adapter-vscode`, `@aafe/adapter-codebuddy`.
